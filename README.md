# AppMedical

Desarrollo de una App Medical con Spring Boot 3

## Objetivos.

- Creacion de una Api Rest
- Crud (Create, Read, Update, Delete)
- Validaciones de datos
- Paginacion y Ordenamiento

## Tecnologias Utilizadas.

- Spring Boot 3
- Java 17
- Lombok
- MySQL Server + Flaway
- JPA / Hibernate
- Maven
- Postman
